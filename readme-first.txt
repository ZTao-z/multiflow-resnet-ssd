本project为发表于neurocomputing的文章对应代码，具体与实验章节四个小节一一对应。
在运行每个小节对应的实验代码时，需要将该小节文件夹中的训练代码放到根目录下，将模型代码放入models文件夹中，再运行代码，

举例，如要运行section4.1中输入图像尺寸为300*300的代码，需要将train_mssd_07+12.py放到根目录，然后将
res_34.py放入models文件夹，然后运行train_mssd+07+12.py代码。

注1：其他相关参数设置（如预先设定的anchor box个数、尺寸等）可于data文件夹中的config.py中设置。
注2：section4.3中的输入尺寸为512*512的训练代码可根据输入为300*300的训练代码直接改动，并再config.py中进行相关参数改动，
具体改动细节可参考论文以及github上RFB-Net的project。
注3：section4.4中各个消融实验，可直接对res_34.py进行改动后进行实验，即注释掉对应的组件即可。
